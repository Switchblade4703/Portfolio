Shiny hunter

The Shiny hunter is an application that started as a group project but i brought it solo so i could keep working on it to develop a basic ui and fix a couple of bugs

It is made in python and uses simulated hardware keypresses and machine vision to play pokemon for you so you can do something else while it does the mundane task of shiny hunting for you
